Distributed Training Concepts

мқҙ л¬ём„ңлҠ” мҝЎл¶Ғ м „л°ҳм—җм„ң л°ҳліөн•ҙм„ң л“ұмһҘн•ҳлҠ” н•өмӢ¬ к°ңл…җмқ„ н•ңмһҗлҰ¬м—җ лӘЁм•„ л‘” кёҖмһ…лӢҲлӢӨ. лӢӨлҘё кіім—җм„ң к°ҷмқҖ м„ӨлӘ…мқ„ л°ҳліөн•ҳм§Җ м•Ҡкё° мң„н•ң кіөмҡ© м–ҙнңҳм§‘мқҙлқјкі  ліҙл©ҙ лҗ©лӢҲлӢӨ.

ліё мҝЎл¶Ғмқҙ лӢӨлЈЁлҠ” лі‘л ¬нҷ”: DDP н•ң к°Җм§Җ

л¶„мӮ° н•ҷмҠө м „лһөмқҖ м—¬лҹ¬ к°Җм§Җк°Җ мһҲм§Җл§Ң, ліё мҝЎл¶ҒмқҖ DDP(DistributedDataParallel) н•ң к°Җм§Җл§Ң лӢӨлЈ№лӢҲлӢӨ. к°Ғ GPUм—җ лӘЁлҚёмқ„ нҶөм§ёлЎң ліөм ңн•ҙ л‘җкі  лҚ°мқҙн„°лҘј лӮҳлҲ  мІҳлҰ¬н•ҳлҠ” к°ҖмһҘ кё°ліём Ғмқё лҚ°мқҙн„° лі‘л ¬ л°©мӢқмһ…лӢҲлӢӨ.

н•ӯлӘ© DDP (DistributedDataParallel)
лӘЁлҚё л©”лӘЁлҰ¬ к°Ғ GPUм—җ лӘЁлҚё м „мІҙ ліөм ң
к·ёлһҳл””м–ёнҠё к°Ғ GPU кі„мӮ° нӣ„ AllReduce
лҚ°мқҙн„° rankлі„лЎң лҜёлӢҲл°°м№ҳлҘј лӮҳлҲ  мІҳлҰ¬
нҶөмӢ лҹү мһ‘мқҢ (AllReduce 1нҡҢ/мҠӨн…қ)
м Ғн•© лӘЁлҚё нҒ¬кё° н•ң GPUм—җ лӘЁлҚё + optimizer stateк°Җ л“Өм–ҙк°ҖлҠ” нҒ¬кё°

ліё мҝЎл¶Ғмқҳ кё°мӨҖ лӘЁлҚё(concepts-recommender-baseline.md)мқҖ к°ҖмһҘ нҒ° м„Өм •м—җм„ңлҸ„ мҲҳ GB к·ңлӘЁмқҳ мһ„лІ л”© н…Ңмқҙлё”кіј мһ‘мқҖ MLPлЎң кө¬м„ұлҗҳм–ҙ лӢЁмқј GPUм—җ л¬ҙлҰ¬ м—Ҷмқҙ л“Өм–ҙк°‘лӢҲлӢӨ. лӘЁлҚёмқ„ GPU мӮ¬мқҙм—җ мӘјк°ңм•ј н•  л§ҢнҒј нҒ¬м§Җ м•ҠлӢӨлҠ” лң»мқҙл©°, л”°лқјм„ң DDPл§ҢмңјлЎң м¶©л¶„н•©лӢҲлӢӨ. лӘЁлҚё мһҗмІҙлҘј мғӨл”©н•ҙм•ј н•ҳлҠ” FSDPлҠ” мқҙ мҝЎл¶Ғмқҳ лІ”мң„ л°–мқҙлқј лӢӨлЈЁм§Җ м•ҠмҠөлӢҲлӢӨ.

нҶ нҸҙлЎңм§Җ(1Г—1 / 1Г—N / MГ—N)мҷҖ DDPлҘј нҳјлҸҷн•ҳкё° мүҪм§Җл§Ң л‘ҳмқҖ мёөмң„к°Җ лӢӨлҰ…лӢҲлӢӨ. нҶ нҸҙлЎңм§ҖлҠ” GPUлҘј лӘҮ к°ң м–ҙл–»кІҢ л¬¶мқ„м§Җмқҳ л¬ём ңмқҙкі , DDPлҠ” к·ё GPUл“Өм—җ к°ҷмқҖ лӘЁлҚёмқ„ ліөм ңн•ҙ лҚ°мқҙн„°лҘј лі‘л ¬ мІҳлҰ¬н•ҳлҠ” м „лһөмһ…лӢҲлӢӨ. 1Г—Nмқҙл“  MГ—Nмқҙл“  н•ҷмҠө м „лһөмқҖ лӘЁл‘җ DDPмқҙл©°, м°ЁмқҙлҠ” GPUк°Җ н•ң л…ёл“ң м•Ҳм—җ лӘЁм—¬ мһҲлҠҗлғҗ, л…ёл“ңлҘј к°ҖлЎңм§ҖлҘҙлҠҗлғҗлҝҗмһ…лӢҲлӢӨ.

TorchDistributor local_mode: driver vs worker

TorchDistributorм—җм„ң к°ҖмһҘ лЁјм Җ л¶Җл”ӘнһҲлҠ” м„Өм •мқҙ local_mode н”Ңлһҳк·ёмһ…лӢҲлӢӨ. мқҙлҰ„мқҙ лӢӨмҶҢ лӘЁнҳён•ҳм§Җл§Ң мӢӨм ңлЎңлҠ” л¶„мӮ° н•ҷмҠө н”„лЎңм„ёмҠӨлҘј м–ҙл””м—җ лқ„мҡё кІғмқёк°Җ вҖ” driver л…ёл“ң м•Ҳмқём§Җ, Spark worker л…ёл“ңл“Өм—җ л¶„мӮ°н•  кІғмқём§ҖлҘј кІ°м •н•©лӢҲлӢӨ.

local_mode child н”„лЎңм„ёмҠӨк°Җ лҸ„лҠ” кіі мӮ¬мҡ© нҶ нҸҙлЎңм§Җ driverмқҳ м—ӯн• 
True driver л…ёл“ң м•Ҳм—җм„ң Nк°ң н”„лЎңм„ёмҠӨ spawn 1Г—1, 1Г—N (single node) driverк°Җ кі§ н•ҷмҠө л…ёл“ң. driverмқҳ GPUлҘј к·ёлҢҖлЎң мӮ¬мҡ©.
False Spark worker л…ёл“ңл“Өм—җ л¶„мӮ° spawn MГ—N (multi node) driverлҠ” мҪ”л””л„Өмқҙм…ҳл§Ң. н•ҷмҠөм—җлҠ” м°ём—¬н•ҳм§Җ м•ҠмқҢ.

н‘ңлҘј н’Җм–ҙ м“°л©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • local_mode=TrueлҠ” вҖңdriverм—җм„ң н•ҷмҠөн•ңлӢӨвҖқлҠ” лң»мңјлЎң мқҪмңјл©ҙ лҗ©лӢҲлӢӨ. Single-node нҒҙлҹ¬мҠӨн„°(driverл§Ң мһҲкі  worker=0)лӮҳ driverм—җ GPUк°Җ л¶ҷм–ҙ мһҲлҠ” нҒҙлҹ¬мҠӨн„°м—җм„ң мӮ¬мҡ©н•©лӢҲлӢӨ. TorchDistributorк°Җ driver лЁёмӢ  м•Ҳм—җм„ң num_processesк°ңмқҳ Python н”„лЎңм„ёмҠӨлҘј лқ„мҡ°кі  NCCLлЎң л¬¶м–ҙ мӨҚлӢҲлӢӨ.
  • local_mode=FalseлҠ” л°ҳлҢҖлЎң вҖңworkerм—җм„ң н•ҷмҠөн•ңлӢӨвҖқлҠ” лң»мһ…лӢҲлӢӨ. Sparkк°Җ worker л…ёл“ңл“Өм—җ child н”„лЎңм„ёмҠӨлҘј л¶„л°°н•ҳкі , driverлҠ” н•ҷмҠө мһҗмІҙм—җлҠ” лҒјм§Җ м•Ҡкі  RPC мҪ”л””л„Өмқҙм…ҳл§Ң лӢҙлӢ№н•©лӢҲлӢӨ. мқҙ м°ЁмқҙлҠ” лӘЁлӢҲн„°л§Ғм—җлҸ„ к·ёлҢҖлЎң мҳҒн–Ҙмқ„ мЈјлҠ”лҚ°, multi-nodeм—җм„ң driver-side log_system_metrics=Trueл§Ң мјңл©ҙ driverлҠ” н•ҷмҠөмқ„ н•ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ idle л©”нҠёлҰӯл§Ң кё°лЎқлҗ©лӢҲлӢӨ (ops-mlflow-tracking.md В§1).

ліё мҝЎл¶Ғмқҳ к°Ғ нҶ нҸҙлЎңм§Җк°Җ мң„ л‘җ лӘЁл“ң мӨ‘ л¬ҙм—Үм—җ н•ҙлӢ№н•ҳлҠ”м§Җ м •лҰ¬н•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

м…Җ нҳём¶ң мқҳлҜё
1Г—1 TorchDistributor(num_processes=1, local_mode=True) driverм—җм„ң 1к°ң н”„лЎңм„ёмҠӨ. world_size=1мқҙлқј DDP all_reduceлҠ” no-op.
1Г—N TorchDistributor(num_processes=N, local_mode=True) driverмқҳ Nк°ң GPUм—җ Nк°ң н”„лЎңм„ёмҠӨ.
MГ—N TorchDistributor(num_processes=M*N, local_mode=False) Mк°ң worker л…ёл“ңм—җ мҙқ M*N н”„лЎңм„ёмҠӨ л¶„л°°. driverлҠ” мҪ”л””л„Өмқҙм…ҳл§Ң.

мқҙ н”Ңлһҳк·ёлҘј мһҳлӘ» л§һм¶”л©ҙ н•ҷмҠө мһҗмІҙк°Җ мӢңмһ‘лҗҳм§Җ м•Ҡкұ°лӮҳ, мӢңмһ‘лҗҳлҚ”лқјлҸ„ мқҳлҸ„мҷҖ м „нҳҖ лӢӨлҘё лӘЁм–‘мңјлЎң лҸҢм•„к°‘лӢҲлӢӨ. мһҗмЈј л§ҲмЈјм№ҳлҠ” н•Ём •мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • Single-node нҒҙлҹ¬мҠӨн„°м—җм„ң local_mode=FalseлЎң нҳём¶ңн•ҳл©ҙ л¶„л°°н•  workerк°Җ м—Ҷм–ҙ к·ёлҢҖлЎң мӢӨнҢЁн•©лӢҲлӢӨ.
  • Multi-node нҒҙлҹ¬мҠӨн„°м—җм„ң local_mode=TrueлЎң нҳём¶ңн•ҳл©ҙ л№„мӢј worker л…ёл“ңл“Өмқҙ лҶҖкі  driver н•ң лҢҖм—җм„ңл§Ң н•ҷмҠөмқҙ м§„н–үлҗ©лӢҲлӢӨ.
  • local_mode=FalseлЎң лқ„мҡҙ child н”„лЎңм„ёмҠӨлҠ” лӢӨлҘё л…ёл“ңмқҳ fresh Pythonм—җм„ң мӢңмһ‘н•ҳлҜҖлЎң driverмқҳ мһҗкІ©мҰқлӘ…мқҙлӮҳ import мғҒнғңлҘј мһҗлҸҷмңјлЎң мғҒмҶҚл°ӣм§Җ лӘ»н•©лӢҲлӢӨ. мӢӨмҲҳмқҳ лӢЁкіЁ мӣҗмқёмқҙлӢҲ л”°лЎң мӮҙнҺҙ л‘җлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ (debug-common-pitfalls.md В§2, В§2-1).

Lightningм—җлҠ” мҷң local_modeк°Җ м—ҶлҠ”к°Җ вҖ” driverлҠ” м–ҙл–»кІҢ м“°мқҙлҠ”к°Җ

к°ҷмқҖ л¶„мӮ° н•ҷмҠөмқҙлқјлҸ„ Lightningм—җлҠ” local_modeм—җ н•ҙлӢ№н•ҳлҠ” мҳөм…ҳмқҙ ліҙмқҙм§Җ м•ҠмҠөлӢҲлӢӨ. мқҙмң лҠ” лӢЁмҲңн•©лӢҲлӢӨ. local_modeлҠ” TorchDistributorмқҳ launcher мҳөм…ҳмқё л°ҳл©ҙ, PyTorch LightningмқҖ launcherк°Җ м•„лӢҲлқј training frameworkмқҙкё° л•Ңл¬ёмһ…лӢҲлӢӨ. LightningмқҖ л¶„мӮ° нҷҳкІҪ мһҗмІҙлҠ” мҷёл¶Җ launcher(torchrun, SLURM, TorchDistributor л“ұ)к°Җ мһЎм•„ мӨҖлӢӨлҠ” м „м ң мң„м—җм„ң лҸҷмһ‘н•©лӢҲлӢӨ.

к·ёлһҳм„ң L.Trainerк°Җ л°ӣлҠ” мқёмһҗлҸ„ devices, num_nodes, strategy м •лҸ„лЎң лӢЁмҲңн•©лӢҲлӢӨ. вҖңм–ҙл””(driverлғҗ workerлғҗ)м—җм„ң н”„лЎңм„ёмҠӨлҘј лқ„мҡём§ҖвҖқлҠ” Trainerк°Җ м•Ң н•„мҡ”к°Җ м—Ҷкі , к·ёкІғмқҖ н•ң лӢЁкі„ мң„м—җм„ң н”„лЎңм„ёмҠӨлҘј лқ„мҡ°лҠ” launcherмқҳ лӘ«мһ…лӢҲлӢӨ.

кІ°кіјм ҒмңјлЎң driverлҘј н•ҷмҠөм—җ м“°лҠҗлғҗ л§ҲлҠҗлғҗлҠ” launcherк°Җ м–ҙл–»кІҢ мһЎм•„ мЈјлҠҗлғҗм—җ л”°лқј к°ҲлҰҪлӢҲлӢӨ.

мӢңлӮҳлҰ¬мҳӨ н”„лЎңм„ёмҠӨлҘј лқ„мҡ°лҠ” мЈјмІҙ driver мӮ¬мҡ©
л…ёнҠёл¶Ғм—җм„ң Trainer.fit() м§Ғм ‘ нҳём¶ң (single-node) Lightningмқҙ л…ёнҠёл¶Ғ н”„лЎңм„ёмҠӨ мһҗмІҙм—җм„ң child 1~Nк°ң spawn O вҖ” driverмқҳ GPU мӮ¬мҡ©
TorchDistributor(local_mode=True) + Lightning TorchDistributorк°Җ driver м•Ҳм—җм„ң Nк°ң н”„лЎңм„ёмҠӨ spawn, к°Ғмһҗк°Җ Trainer.fit() нҳём¶ң O
TorchDistributor(local_mode=False) + Lightning Sparkк°Җ worker л…ёл“ңл“Өм—җ л¶„мӮ° spawn X вҖ” driverлҠ” мҪ”л””л„Өмқҙм…ҳл§Ң
л…ёнҠёл¶Ғм—җм„ң Trainer(num_nodes=M>1) м§Ғм ‘ нҳём¶ң LightningмқҖ worker л…ёл“ңм—җ н”„лЎңм„ёмҠӨлҘј лқ„мҡё мҲҳлӢЁмқҙ м—ҶмқҢ мӢӨнҢЁ

ліё мҝЎл¶Ғмқҳ Lightning м…Җ(01-notebook-based/04-train_pytorch_lightning_1x1_1xN.ipynb, 05-train_pytorch_lightning_MxN.ipynb)мқҖ мң„ н‘ң мӨ‘ 1В·2В·3н–үм—җ м°ЁлЎҖлЎң л§Өн•‘лҗ©лӢҲлӢӨ.

Lightning лӢЁлҸ…мңјлЎңлҸ„ single-process н•ҷмҠөмқҙ мҷ„кІ°лҗҳкё° л•Ңл¬ём—җ, launcherлҘј н•ң кІ№ лҚ” л‘җлҠ” к°Җм№ҳк°Җ кұ°мқҳ м—ҶмҠөлӢҲлӢӨ. л°ҳл©ҙ 1Г—Nл¶Җн„°лҠ” TorchDistributorк°Җ н”„лЎңм„ёмҠӨ мҠӨнҸ°кіј NCCL нҷҳкІҪ ліҖмҲҳ(RANK, WORLD_SIZE, MASTER_ADDR л“ұ)лҘј мқјкҙҖлҗҳкІҢ мһЎм•„ мЈјлҠ” мқҙм җмқҙ л¶„лӘ…н•ҳлҜҖлЎң к°җмӢё мӨҚлӢҲлӢӨ. MГ—NмқҖ worker л…ёл“ңм—җ н”„лЎңм„ёмҠӨлҘј л¶„л°°н•ҙ мӨ„ мЈјмІҙк°Җ н•„мҡ”н•ҳлҜҖлЎң л°ҳл“ңмӢң local_mode=FalseлЎң к°җмӢём•ј н•©лӢҲлӢӨ (Databricks Blog: PyTorch on Databricks вҖ“ Introducing the Spark PyTorch Distributor).

Databricks кіөмӢқ лё”лЎңк·ёлҠ” multi-node Lightningм—җм„ң Trainer(devices=1, num_nodes=num_processes)лҸ„ к¶ҢмһҘ нҢЁн„ҙмңјлЎң мҶҢк°ңн•©лӢҲлӢӨ. к°Ғ TorchDistributor н”„лЎңм„ёмҠӨлҘј лҸ…лҰҪлҗң вҖңnodeвҖқлЎң м·Ёкёүн•ҙ Lightningмқҙ м¶”к°Җ childлҘј spawnн•ҳм§Җ м•ҠкІҢ л§Ңл“ңлҠ” лӢЁмҲңнҷ” нҢЁн„ҙмһ…лӢҲлӢӨ. ліё мҝЎл¶ҒмқҖ нҒҙлҹ¬мҠӨн„° нҶ нҸҙлЎңм§ҖлҘј к·ёлҢҖлЎң л“ңлҹ¬лӮҙл ӨлҠ” мқҳлҸ„лЎң devices=N, num_nodes=Mмқ„ мӮ¬мҡ©н•©лӢҲлӢӨ. л‘ҳ лӢӨ лҸҷмһ‘н•ҳм§Җл§Ң мқҳлҜёмҷҖ л””лІ„к№… кҙҖм җмқҙ лӢӨлҘҙлӢӨлҠ” м җл§Ң м•Ңм•„ л‘җл©ҙ лҗ©лӢҲлӢӨ.

Lightningмқ„ TorchDistributorмҷҖ мЎ°н•©н•  л•Ң мһҗмЈј л§ҢлӮҳлҠ” н•Ём •лҸ„ м •лҰ¬н•ҙ л‘ЎлӢҲлӢӨ.

  • л…ёнҠёл¶Ғм—җм„ң Trainer(num_nodes=M>1)лҘј TorchDistributor м—Ҷмқҙ м§Ғм ‘ нҳём¶ңн•ҳл©ҙ Lightningмқҙ worker л…ёл“ңм—җ м ‘к·јн•  мҲҳлӢЁмқҙ м—Ҷм–ҙ к·ёлҢҖлЎң л©Ҳм¶ҘлӢҲлӢӨ. Databricks нҷҳкІҪм—җм„ң multi-node LightningмқҖ л°ҳл“ңмӢң TorchDistributor(local_mode=False)лЎң к°җмӢём•ј н•©лӢҲлӢӨ.
  • л°ҳлҢҖлЎң TorchDistributor(local_mode=True, num_processes=N) м•ҲмӘҪм—җм„ң Trainer(num_nodes>1)мқ„ мЈјлҠ” мЎ°н•©лҸ„ н”јн•ҙм•ј н•©лӢҲлӢӨ. TorchDistributorлҠ” driver н•ң лҢҖ м•Ҳм—җм„ң Nк°ң н”„лЎңм„ёмҠӨлҘј лқ„мҡ°лҠ”лҚ°, Lightningм—җлҠ” вҖңм—¬лҹ¬ л…ёл“ңвҖқлқјкі  кұ°м§“л§җмқ„ н•ҳлҠ” м…Ҳмқҙ лҗҳм–ҙ rendezvousк°Җ м–ҙкёӢлӮ©лӢҲлӢӨ.
  • TorchDistributorмқҳ child н”„лЎңм„ёмҠӨлҠ” л…ёнҠёл¶Ғкіј лі„к°ңмқё fresh Pythonм—җм„ң мӢңмһ‘н•©лӢҲлӢӨ. к·ёлһҳм„ң LightningModule, DataModule, callback к°ҷмқҖ к°қмІҙлҠ” train_fn м•Ҳм—җм„ң лӢӨмӢң м •мқҳн•ҳкұ°лӮҳ importн•ҙ мЈјм–ҙм•ј н•©лӢҲлӢӨ (debug-common-pitfalls.md В§2).

мҷң multi-node DDPмқёк°Җ (MГ—N м…Җмқҳ м •лӢ№нҷ”)

м¶”мІң лӘЁлҚёмҡ© MLPлҠ” лӢЁмқј GPUм—җлҸ„ м¶©л¶„нһҲ л“Өм–ҙк°ҖлҠ”лҚ° көімқҙ multi-node DDPк№Ңм§Җ лӢӨлЈЁлҠ” мқҙмң к°Җ к¶ҒкёҲн•  мҲҳ мһҲмҠөлӢҲлӢӨ. ліё мҝЎл¶Ғмқҙ MГ—N м…Җмқ„ л”°лЎң л§Ҳл Ён•ҙ л‘” мқҙмң лҠ” л‘җ к°Җм§Җмһ…лӢҲлӢӨ.

  1. лҚ°мқҙн„° мІҳлҰ¬лҹү нҷ•мһҘ. user-item interactionмқҙ мҲҳмӢӯм–ө н–ү к·ңлӘЁлЎң м»Өм§Җл©ҙ, н•ң л…ёл“ңл§ҢмңјлЎңлҠ” н•ң м—җнҸӯмқ„ лҸ„лҠ” лҚ° л„Ҳл¬ҙ мҳӨлһҳ кұёлҰҪлӢҲлӢӨ. л…ёл“ңлҘј лҠҳлҰ¬л©ҙ мІҳлҰ¬лҹүмқ„ кұ°мқҳ л…ёл“ң мҲҳм—җ л№„лЎҖн•ҙ лҒҢм–ҙмҳ¬лҰҙ мҲҳ мһҲмҠөлӢҲлӢӨ.
  2. TorchDistributor мӮ¬мҡ©лІ• мӢңм—°. Databricksм—җм„ң multi-node л¶„мӮ° launcherлҘј лқ„мҡ°лҠ” нҢЁн„ҙ мһҗмІҙлҘј, мқөмҲҷн•ң лӘЁлҚё кіЁкІ© мң„м—җм„ң ліҙм—¬ мЈјл ӨлҠ” мқҳлҸ„мһ…лӢҲлӢӨ.

лӢӨл§Ң лӘЁлҚёмқҙ мһ‘мқ„мҲҳлЎқ AllReduce нҶөмӢ  л№„мҡ©мқҙ мғҒлҢҖм ҒмңјлЎң м»Өм§‘лӢҲлӢӨ. л…ёл“ңлҘј л¬ҙмһ‘м • лҠҳлҰ¬л©ҙ throughputмқҙ л№„лЎҖн•ҙм„ң лҠҳм§Җ м•ҠлҠ” кө¬к°„мқҙ кёҲм„ё м°ҫм•„мҳөлӢҲлӢӨ. ліё мҝЎл¶Ғмқҳ MГ—N м…ҖмқҖ вҖңнҢЁн„ҙмқҙ лҸҷмһ‘н•ңлӢӨвҖқлҠ” кІғмқ„ ліҙмқҙлҠ” лҚ° мҙҲм җмқ„ л§һм¶”л©°, ліёкІ©м Ғмқё мҠӨмјҖмқјл§Ғ мӢӨн—ҳмқҖ мӮ¬мҡ©мһҗ мӣҢнҒ¬лЎңл“ң нҠ№м„ұм—җ л§һм¶° лі„лҸ„лЎң м§„н–үн•ҳмӢңлҠ” кІғмқ„ к¶ҢмһҘн•©лӢҲлӢӨ.

rank к°„ metric н•©мӮ° (AverageMeter)

DDP нҷҳкІҪм—җм„ң кІҖмҰқ lossлӮҳ accuracyлҘј мёЎм •н•  л•Ң нқ”нһҲ л№ лңЁлҰ¬лҠ” л¶Җл¶„мһ…лӢҲлӢӨ. к°Ғ rankлҠ” мһҗкё° лӘ«мқҳ лҚ°мқҙн„°л§Ң ліҙкё° л•Ңл¬ём—җ, rankлі„ нҸүк· мқ„ к·ёлҢҖлЎң м¶ңл Ҙн•ҳл©ҙ м „мІҙ кІҖмҰқ м…Ӣ нҸүк· кіј м–ҙкёӢлӮ©лӢҲлӢӨ. мҳ¬л°”лҘё м „мІҙ нҸүк· мқ„ кө¬н•ҳл Өл©ҙ dist.all_reduce(SUM)мңјлЎң rankлі„ н•©кіј м№ҙмҡҙнҠёлҘј лӘЁмқҖ л’Ө driver(rank 0)м—җм„ң лӮҳлҲ м•ј н•©лӢҲлӢӨ. ліё мҝЎл¶ҒмқҖ лӢӨмқҢкіј к°ҷмқҖ AverageMeter нҢЁн„ҙмқ„ н‘ңмӨҖмңјлЎң мӮ¬мҡ©н•©лӢҲлӢӨ.

import torch
import torch.distributed as dist


class AverageMeter:
    """rankлі„ лҲ„м Ғм№ҳлҘј л“Өкі  мһҲлӢӨк°Җ all_reduceлЎң м „мІҙ нҸүк· мқ„ л§Ңл“ӯлӢҲлӢӨ."""

    def __init__(self):
        self.reset()

    def reset(self):
        self.loss_sum = 0.0
        self.correct = 0
        self.total = 0

    def update(self, loss_sum: float, correct: int, total: int):
        self.loss_sum = loss_sum
        self.correct = correct
        self.total = total

    def all_reduce(self, device) -> tuple[float, float]:
        """лӘЁл“  rankмқҳ (loss_sum, correct, total)мқ„ н•©мӮ°н•ҳкі  нҸүк· /м •нҷ•лҸ„лҘј л°ҳнҷҳ."""
        t = torch.tensor([self.loss_sum, self.correct, self.total], device=device)
        dist.all_reduce(t, op=dist.ReduceOp.SUM, async_op=False)
        loss_sum, correct, total = t.tolist()
        return loss_sum / total, correct / total

    def reduce(self) -> tuple[float, float]:
        """non-distributed м»Ён…ҚмҠӨнҠёмҡ© fallback."""
        return self.loss_sum / self.total, self.correct / self.total

meter = AverageMeter()
loss_sum, correct, total = 0.0, 0, 0
