лқјмқҙлёҢлҹ¬лҰ¬ кҙҖлҰ¬

л¶„мӮ° н•ҷмҠөм—җм„ң к°ҖмһҘ мһҗмЈј мӮ¬кі к°Җ лӮҳлҠ” м§Җм җмқҖ вҖңdriverм—җм„ңлҠ” import лҗҳлҠ”лҚ° worker childм—җм„ңлҠ” м•Ҳ лҗЁвҖқмһ…лӢҲлӢӨ. DatabricksлҠ” лқјмқҙлёҢлҹ¬лҰ¬ м„Өм№ҳ кІҪлЎңк°Җ м—¬лҹ¬ к°ңлқјм„ң, м–ҙл””м—җ м–ҙл–»кІҢ м„Өм№ҳн•ҳлҠҗлғҗк°Җ childм—җм„ңмқҳ к°ҖмӢңм„ұмқ„ кІ°м •н•©лӢҲлӢӨ.

3к°Җм§Җ м„Өм№ҳ кІҪлЎң

м„ё к°Җм§Җ м„Өм№ҳ кІҪлЎңмқҳ м Ғмҡ© лІ”мң„мҷҖ child к°ҖмӢңм„ұмқ„ л№„көҗн•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

кІҪлЎң лӘ…л № м Ғмҡ© лІ”мң„ child к°ҖмӢңм„ұ
Notebook-scoped %pip install + %restart_python л…ёнҠёл¶Ғмқҙ attachлҗң clusterмқҳ driverмҷҖ лӘЁл“  workerмқҳ нҳ„мһ¬ Python env TorchDistributor childлҸ„ к°ҷмқҖ envм—җм„ң мӢңмһ‘н•ҳлҜҖлЎң ліҙмһ„
Cluster library UI > Libraries нғӯ лҳҗлҠ” cluster JSON cluster мӢңмһ‘ мӢң driver + workerм—җ м„Өм№ҳ ліҙмһ„
Init script cluster мӢңмһ‘ мӢң мӢӨн–үлҗҳлҠ” shell script driver + worker ліҙмһ„ (%pipліҙлӢӨ л№ лҘё лӢЁкі„м—җм„ң м„Өм№ҳ)

ліё мҝЎл¶ҒмқҖ Notebook-scoped %pipлҘј кё°ліёмңјлЎң мӮ¬мҡ©н•©лӢҲлӢӨ(01-notebook-based/00-setup.ipynbмқҳ %pip install --quiet "lightning==2.5.1" "nvidia-ml-py").

%pip installмқҙ childм—җ ліҙмқҙлҠ” мқҙмң 

м§ҒкҙҖм—җ л°ҳн•ҳм§Җл§Ң %pip installмқҖ лӢЁмҲңнһҲ вҖңнҳ„мһ¬ л…ёнҠёл¶Ғм—җм„ңл§Ң ліҙмқҙлҠ” venvвҖқк°Җ м•„лӢҷлӢҲлӢӨ. мӢӨм ңлЎңлҠ” лӢӨмқҢ лӢЁкі„лҘј кұ°м№©лӢҲлӢӨ.

  1. driverмқҳ attached Python envм—җ нҢЁнӮӨм§ҖлҘј м„Өм№ҳн•©лӢҲлӢӨ.
  2. clusterмқҳ лӘЁл“  workerм—җлҸ„ лҸҷкё°н•©лӢҲлӢӨ(Databricksк°Җ backgroundм—җм„ң мІҳлҰ¬).
  3. %restart_pythonмңјлЎң driverмқҳ Python interpreterлҘј мһ¬мӢңмһ‘н•ҙ мғҲ нҢЁнӮӨм§ҖлҘј importн•  мҲҳ мһҲкІҢ л§Ңл“ӯлӢҲлӢӨ.

к·ёлһҳм„ң TorchDistributorк°Җ лқ„мҡ°лҠ” worker child н”„лЎңм„ёмҠӨлҠ” к°ҷмқҖ clusterмқҳ attached Python envм—җм„ң fresh interpreterлЎң мӢңмһ‘н•ҳлҜҖлЎң %pip installн•ң нҢЁнӮӨм§Җк°Җ мһҗлҸҷмңјлЎң ліҙмһ…лӢҲлӢӨ. лӢӨмқҢ мҪ”л“ңк°Җ к·ё лҸҷмһ‘мқ„ нҷ•мқён•ҳлҠ” к°ҖмһҘ м§§мқҖ мҳҲмһ…лӢҲлӢӨ.

# driver
%pip install some-package
%restart_python

# worker child (TorchDistributor.runмңјлЎң лқ„мӣҖ)
def fn():
    import some_package        # OK вҖ” к°ҷмқҖ env
    print(some_package.__version__)

н•Ём • 1: %restart_python лҜёмӢӨн–ү

%pip install нӣ„м—җ %restart_pythonмқ„ л№јлЁ№мңјл©ҙ лӢӨмқҢкіј к°ҷмқҙ м–ҙкёӢлӮң мғҒнғңк°Җ лҗ©лӢҲлӢӨ.

  • мғҲ нҢЁнӮӨм§ҖлҠ” л””мҠӨнҒ¬м—җ м„Өм№ҳлҗҳм–ҙ мһҲм§Җл§Ң нҳ„мһ¬ Python interpreterм—җлҠ” л°ҳмҳҒлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.
  • л°ҳл©ҙ TorchDistributor childлҠ” fresh interpreterлЎң мӢңмһ‘н•ҳлҜҖлЎң к·ёмӘҪм—җм„ңлҠ” ліҙмһ…лӢҲлӢӨ.

к°ҷмқҖ л…ёнҠёл¶Ғм—җм„ңлҠ” importк°Җ мӢӨнҢЁн•ҳлҠ”лҚ° childм—җм„ңлҠ” м„ұкіөн•ҳлҠ” к°ҖмһҘ н—·к°ҲлҰ¬лҠ” мҰқмғҒмқҙ м—¬кё°м—җм„ң лӮҳмҳөлӢҲлӢӨ. %pip install м§Ғнӣ„м—җлҠ” н•ӯмғҒ %restart_pythonмқ„ н•Ёк»ҳ мӢӨн–үн•ҳм„ёмҡ”.

н•Ём • 2: cluster restart нӣ„ %pip нңҳл°ң

%pip installмқҖ clusterмқҳ лқјмқҙлёҢлҹ¬лҰ¬ мғҒнғңм—җ runtime лҸҷм•Ҳл§Ң лӮЁкі , clusterлҘј restartн•ҳл©ҙ мӮ¬лқјм§‘лӢҲлӢӨ. мҳҒн–ҘмқҖ лӢӨмқҢкіј к°ҷмқҙ лӮҳнғҖлӮ©лӢҲлӢӨ.

  • л§Ө м„ём…ҳл§ҲлӢӨ 00-setup.ipynbмқҳ %pip м…Җмқ„ лӢӨмӢң мӢӨн–үн•ҙм•ј н•©лӢҲлӢӨ.
  • JobмңјлЎң лқ„мҡё л•ҢлҠ” setup л…ёнҠёл¶Ғмқ„ лЁјм Җ мӢӨн–үн•ҙ лҸҷмқј clusterм—җ нҢЁнӮӨм§Җк°Җ мӮҙм•„ мһҲлҸ„лЎқ ліҙмһҘн•ҙм•ј н•©лӢҲлӢӨ.

лҢҖм•ҲмңјлЎң cluster library(UI) лҳҗлҠ” init scriptлЎң л“ұлЎқн•ҳл©ҙ cluster restart нӣ„м—җлҸ„ мҳҒкө¬м ҒмңјлЎң мң м§Җлҗ©лӢҲлӢӨ.

н•Ём • 3: мқҳмЎҙм„ұ м¶©лҸҢ

DBR 17.3 LTS MLмқҖ torch, accelerate, transformers л“ұмқ„ кі м • лІ„м „мңјлЎң мӮ¬м „ м„Өм№ҳн•©лӢҲлӢӨ(env-databricks-environments.md н‘ң м°ёмЎ°). %pip install lightning к°ҷмқҖ лӘ…л №мқҖ lightningмқҙ мҡ”кө¬н•ҳлҠ” torch лІ„м „мқҙ DBR мӮ¬м „ м„Өм№ҳ лІ„м „кіј нҳёнҷҳлҗҳм§Җ м•Ҡмңјл©ҙ torchлҘј downgradeВ·upgradeн•ҳл Ө мӢңлҸ„н•ҳкі , к·ё кіјм •м—җм„ң NCCLВ·CUDA нҳёнҷҳм„ұмқҙ к№Ём§Ҳ мҲҳ мһҲмҠөлӢҲлӢӨ.

к¶ҢмһҘ нҢЁн„ҙмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • %pip install --quiet "lightning==2.5.1"мІҳлҹј DBRкіј нҳёнҷҳлҗҳлҠ” м •нҷ•н•ң лІ„м „мқ„ кі м •н•©лӢҲлӢӨ(ліё мҝЎл¶Ғ нҢЁн„ҙ).
  • нҒ° нҢЁнӮӨм§Җ(torch, transformers)лҠ” DBRмқҙ м ңкіөн•ҳлҠ” кІғмқ„ к·ёлҢҖлЎң мӮ¬мҡ©н•©лӢҲлӢӨ.
  • мӢ к·ң м¶”к°ҖлҠ” мөңмҶҢнҷ”н•©лӢҲлӢӨ. ліё мҝЎл¶ҒмқҖ lightningкіј nvidia-ml-pyл§Ң м¶”к°Җн•©лӢҲлӢӨ.

н•Ём • 4: childк°Җ system Pythonмқ„ лҙ„

02-script-based/08-launch_accelerator_MxN.ipynbмқҳ н•өмӢ¬ н•Ём •мһ…лӢҲлӢӨ. subprocess.Popen(["accelerate", ...])лЎң м§Ғм ‘ нҳём¶ңн•ҳл©ҙ PATHм—җ мһЎнһҢ /databricks/python3/bin/accelerateк°Җ мӮ¬мҡ©лҗҳм–ҙ system Python envм—җм„ң мӢӨн–үлҗ©лӢҲлӢӨ. к·ё envлҠ” notebook-scoped %pipк°Җ м„Өм№ҳн•ң нҢЁнӮӨм§ҖлҘј ліҙм§Җ лӘ»н•©лӢҲлӢӨ.

03 н–үмқҳ нҢЁнӮӨм§Җ installмқҙ к°ҖмһҘ мӨ‘мҡ”н•ң мқҙмң к°Җ м—¬кё°м—җ мһҲмҠөлӢҲлӢӨ. recommender_pkgлҠ” wheelлЎң notebook-scoped envм—җ м„Өм№ҳлҗҳлҜҖлЎң, accelerateк°Җ system Pythonм—җм„ң лҸ„лҠ” н•ң importлҠ” мӢӨнҢЁн•©лӢҲлӢӨ.

03-custom-package-script-based/08-launch_accelerator_MxN.ipynbк°Җ мұ„нғқн•ң н•ҙкІ°мұ…мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

import sys
inference_cmd = f"{sys.executable} -m accelerate.commands.accelerate_cli launch -m recommender_pkg.torch_distributor_trainer ..."

мқҙ лӘ…л №мқҙ лҸҷмһ‘н•ҳлҠ” мқҙмң лҘј н’Җм–ҙ м“°л©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • sys.executableмқҖ notebook-scoped Pythonмқҳ м ҲлҢҖ кІҪлЎңмһ…лӢҲлӢӨ.
  • -m accelerate.commands.accelerate_cliлЎң лӘЁл“Ҳ лӘЁл“ң нҳём¶ңмқҙ лҗҳм–ҙ, notebook envм—җм„ң accelerateлҘј importн•©лӢҲлӢӨ.
  • -m recommender_pkg.torch_distributor_trainerлЎң entry pointлҸ„ лӘЁл“Ҳ лӘЁл“ңлЎң нҳём¶ңн•ҳл©ҙ wheelм—җ м„Өм№ҳлҗң нҢЁнӮӨм§ҖлҘј м •мғҒм ҒмңјлЎң importн•  мҲҳ мһҲкі , sys.path[0]мқҙ нҢЁнӮӨм§Җ лӮҙл¶ҖлЎң мһҳлӘ» мһЎнһҲлҠ” л¬ём ңлҸ„ н•Ёк»ҳ нҡҢн”јлҗ©лӢҲлӢӨ.
